    Overview

    The Kenya National Phage Bank (KNPB) is a proposed digital and physical repository for bacteriophage data. Built for students, researchers, and national AMR programs, the system integrates structured metadata, genome sequences, and AI models to predict host range and guide therapeutic phage discovery.

    Workflow Diagram

    This diagram shows how students will upload samples, how data flows through the system, and how machine learning and AMR surveillance provide insights for phage discovery.

    Key Features

    • Metadata Capture: Phage ID, GPS location, host species, plaque morphology.
    • Sequence Storage: FASTA genomes, GenBank annotations, GC content.
    • AI Predictions: Host range, phenotype estimates, clustering.
    • AMR Linkage: Uses aggregated NDAR data to match phages to resistant pathogens.
    • Student Workflow: Upload → QC → Extract → ML → Dashboard → Repository.
